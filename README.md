# Amir Khan - Portfolio Website

A professional personal portfolio website for Amir Khan, an AI-Driven Business Analyst and Data Professional. Built with a modern glass-gradient design aesthetic and responsive bento-grid layout.

## Features

- **Modern Design**: Glass-gradient aesthetic with translucent cards and smooth animations
- **Responsive Layout**: Fully responsive design that works on all devices
- **Multi-page Structure**: 7 pages including Home, About, Projects, Skills, Certificates, Resume, and Contact
- **Smooth Animations**: Scroll-reveal animations and hover effects
- **Professional Content**: Well-crafted copywriting focused on data analytics and business intelligence

## Pages

1. **index.html** - Main landing page with bento-grid layout and hero section
2. **about.html** - About page with professional background and philosophy
3. **projects.html** - Portfolio of data-driven projects and solutions
4. **skills.html** - Comprehensive list of skills and expertise
5. **certificates.html** - Professional certifications and credentials
6. **resume.html** - Professional experience and education timeline
7. **contact.html** - Contact form and social media links

## Design System

- **Primary Background**: #0A0F1C
- **Glass Panels**: rgba(255,255,255,0.06) with backdrop-filter blur
- **Accent Colors**: Cyan (#00E0FF) + Emerald (#00FFA3) gradient
- **Typography**: Outfit (headings) + Inter (body)
- **Animations**: Fade-in on scroll, card hover glow, smooth transitions

## Setup

1. Clone or download this repository
2. Add your profile image to `assets/images/profile.jpg`
3. Add your resume PDF to `assets/pdf/Amir_Khan_Resume.pdf`
4. Add a favicon to `assets/images/favicon.ico`
5. Update contact information and social media links in all HTML files
6. Customize content as needed

## Deployment

This website is ready for deployment on:

- **GitHub Pages**: Simply push to a GitHub repository and enable Pages
- **Vercel**: Connect your repository to Vercel for automatic deployments
- **Netlify**: Drag and drop the folder or connect via Git
