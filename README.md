# Amir Khan - AI-Driven Business Analyst Portfolio

A professional portfolio website for **Amir Khan**, bridging operational experience with AI-augmented analytics. Built with a modern glass-gradient design aesthetic and a strategic narrative focused on business impact.

## Features

- **Modern Design**: Glass-gradient aesthetic with translucent cards and smooth animations.
- **Responsive Layout**: Fully responsive design that works on all devices.
- **Strategic Narrative**: Content designed to psychologically motivate hiring managers.
- **Professional Branding**: "The Practical Analyst" positioning (Operations + AI).

## Site Structure

1.  **Home (`index.html`)**: Executive summary and "Hub" linking to key sections.
2.  **About (`about.html`)**: Professional journey from Field Sales to Data Science.
3.  **Projects (`projects.html`)**: Deep dive into the "BlueMart Retail Analytics" project.
4.  **How I Work (`howiwork.html`)**: The "AI-Augmented" methodology and workflow.
5.  **Learning Journey (`learningjourney.html`)**: Evolution of skills and certification milestones.
6.  **Contact (`contact.html`)**: Professional contact form and direct channels.

## Design System

-   **Primary Background**: `#0A0F1C` (Deep Dark Blue)
-   **Glass Panels**: `rgba(255,255,255,0.03)` with backdrop-filter blur
-   **Accent Colors**: Cyan (`#00E0FF`) + Emerald (`#00FFA3`) gradient
-   **Typography**: Outfit (headings) + Inter (body)

## Deployment

This website is ready for deployment on **GitHub Pages**, **Vercel**, or **Netlify**.
